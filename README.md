# TesloDB API

1. Clonar Proyecto
2. ```yarn install```
3. Clonar achivo ```.env.template``` y renombralo a ```.env```
4. Cambiar las variables de entorno
5. Levantar la bade de datos ```docker-compose up -d```
6. Ejecutar Seed ```localhost:3000/api/seed```
6. Levantar: ```yarn start:dev```
